# Azeroth Explorer

Fly around World of Warcraft's Azeroth in **Google Earth Pro** — maps only. No wargame markers, no scripts, no live updates.

**Version:** 3.0.0 (Azeroth Explorer — Pacific theater)  
**Globe:** v3 · Built 2026-06-21

---

## Download

On GitHub Releases, under **Assets**, download **only**:

- **`Azeroth-Explorer-3.0.0-MAP.zip`**

**Do not** download “Source code (zip)” or “Source code (tar.gz)” — those are the project repo, not the map.

## Install Google Earth Pro

Download and install **Google Earth Pro** from Google's Earth versions page. Launch it before you open the map below.

## Open this map

1. Put all downloaded parts in one folder.
2. Join them (see `HOW_TO_JOIN.txt`), then unzip the joined archive.
3. Keep `Azeroth Explorer.kml`, `kml/`, and `tiles/` together.
4. In Google Earth Pro, choose **File → Open** and select **`Azeroth Explorer.kml`**.

## Navigate

- Expand **Map layers → Quick View** and double-click **Planet** or a continent name to fly there.
- Zoom with the scroll wheel; drag to pan and rotate the globe.
- Map imagery loads as you zoom in. Wait a moment near each landmass while tiles load.

## Updates

Download the latest zip from GitHub Releases: https://github.com/khallammarellus-rgb/Azeroth-Explorer/releases

## What this is not

- Not **WoW Commander** — no campaign board, turn export, or live play layer.
- No internet refresh — everything runs from files on your computer.
